# Another Example

A second placeholder, here to show that more notebooks slot in automatically — no configuration, no edits to the build script.

## Adding Pages Is Free

Drop a new `.md` file in `notebooks/` and it appears in the site. Reorder pages by renaming the numeric prefixes (`01-`, `02-`, `03-`).

## Make It Yours

Delete this file once you have your own notes, or keep it around as a scratch page. Either way, you are only ever editing Markdown.
